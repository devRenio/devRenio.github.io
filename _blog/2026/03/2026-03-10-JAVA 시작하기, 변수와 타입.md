## JAVA프로그래밍I(조성윤) 2주차 강의내용

### Java의 특성

대부분의 운영체제에는 기본적으로 자바 가상 머신(JVM)이 설치되어 있어,<br>
Java 소스 파일(.java)을 컴파일한 바이트코드 파일(.class)를 실행시킬 수 있다.

### 변수와 타입

Java는 포인터가 없다.

문자열 타입은 String.

캐스팅 연산자((type))를 이용하여 더 작은 타입으로도 강제 형변환 가능.
